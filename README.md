# Task-manager-CRUD
A Python-based task management system with CRUD operations, task completion tracking, priority management, dashboard statistics, and persistent file storage.
## Features

* Add Tasks
* View Tasks
* Update Tasks
* Delete Tasks
* Mark Tasks as Completed
* Priority Management
* Dashboard Statistics
* File Storage using Text Files

## Concepts Used

* Object-Oriented Programming (OOP)
* Lists
* CRUD Operations
* File Handling
* Exception Handling
* Loops and Conditionals

## How to Run

```bash
python task-manager-crud.py
```

## Sample Features

1. Add Task
Create a new task.
Assign a priority level:
High
Medium
Low 

2. View Tasks
Display all available tasks.
Show:
Task Title
Priority
Completion Status

3. Update Task
Modify an existing task.
Change task title.
Change task priority.

4. Delete Task
Remove unwanted tasks from the list.

5. Mark Task Complete
Mark pending tasks as completed.
Update task status automatically.

6. Dashboard
View task statistics:
Total Tasks
Completed Tasks
Pending Tasks

7. File Storage
Tasks are automatically saved to a text file.
Data persists even after closing the application.
Previously saved tasks are loaded automatically when the program starts.

### Persistent Storage

Tasks are automatically saved to `tasks.txt` and loaded when the application starts.
